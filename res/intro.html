R-eCommAgent  is an agent-based software designed to allow the signed user to purchase Items on registered web-sites, and receive product recommendations according to his profile and purchasing activities in the system. The system utilizes a distributed platform of agent modules to provide a diverse product recommendation base.
 


Users - The main functionalities

Search Products - Search products on the registered web-sites by category, maximum price or by free-text search

All Recommendations  - See all received recommendations details sorted by date (four latest recommendations are always presented on Latest Recommendations panel)

My Shopping - See all purchased products details sorted by date

My Viewed Products - See all viewed (but not yet purchased) products details sorted by date

People Online - See other details of all the users, who are currently connected to the R-eCommAgent platform

User profile - See and update personal details

R-eCommAgent Menu - Sign - in as different user or exit the program

Help Menu - View help and product description

 

Administrator - The main functionalities

Administrator User, in addition to regular user functionalities has the ability to browse and define system parameters that affect platform-wide settings.

Administration Menu allows to -

View system Reports

Add/Update system Category

Register/Update system Web-site

Update parameters that affect system Recommendation processing

 

For more information about R-eCommAgent  see documentation supplied with the installation.
